Sistema de Controle de Manutenção de Colheitadeiras

Gerencie sua frota com eficiência

Solução completa para o gerenciamento de manutenções preventivas e corretivas, controle de horas de trabalho, trocas de óleo e estoque de peças para colheitadeiras John Deere.

Manutenções Preventivas

Agende e acompanhe manutenções preventivas baseadas em horas de uso, garantindo que suas colheitadeiras estejam sempre em perfeitas condições de operação.

Manutenções Corretivas

Registre falhas, peças substituídas e soluções aplicadas, criando um histórico completo de todas as manutenções corretivas realizadas em cada equipamento.

Controle de Horímetro

Monitore as horas de trabalho de cada colheitadeira, permitindo um planejamento preciso das manutenções e substituição de componentes.

Trocas de Óleo

Registre e acompanhe todas as trocas de óleo, com informações sobre tipo, quantidade e próxima troca programada baseada em horas de uso.

Gestão de Estoque

Controle completo do estoque de peças e produtos, com alertas de níveis mínimos e registro de todas as movimentações de entrada e saída.

Relatórios Detalhados

Gere relatórios detalhados sobre o histórico de manutenções, uso de peças, horas trabalhadas e muito mais para uma gestão eficiente da sua frota.

Instruções de Instalação Simplificadas

Siga estes passos simples para instalar e executar o Sistema de Controle de Manutenção de Colheitadeiras:

  1. Baixe o sistema clicando no botão "Baixar Sistema" acima.
  2. Descompacte o arquivo em uma pasta de sua escolha.
  3. Instale as dependências executando o seguinte comando no terminal:
    pip install -r requirements.txt
  4. Inicialize o banco de dados executando:
    python init_db.py
  5. Inicie a aplicação com o comando:
    python app.py
  6. Acesse o sistema abrindo seu navegador e visitando:
    http://localhost:5000

Credenciais de acesso:

  • Administrador: admin / admin123
  • Técnico: tecnico / tecnico123
  • Operador: operador / operador123

Implantação Online (Opcional)

Para implantar o sistema online e acessá-lo de qualquer lugar, recomendamos o uso da plataforma Render:

  1. Crie uma conta no Render
  2. Clique em "New +" e selecione "Web Service"
  3. Escolha a opção "Upload Files" e faça upload do arquivo ZIP do sistema
  4. Configure o serviço:
    • Nome: sistema-manutencao-colheitadeiras
    • Runtime: Python 3
    • Build Command: pip install -r requirements.txt
    • Start Command: gunicorn app:app
  5. Em "Advanced", adicione as variáveis de ambiente:
    • FLASK_APP: app.py
    • FLASK_ENV: production
    • SECRET_KEY: sistema_manutencao_secret_key
  6. Após a implantação, acesse a seção "Shell" e execute: python init_db.py
